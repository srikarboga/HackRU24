# HackRU24

# Build instructions:

clone this repository

in the frontend directory run "npm install"

in the backend directory make sure to run "pip install -r requirements.txt" to make sure all of the dependencies are up to date

next you can start the backsend server by running "uvicorn backend:app --reload" in the backend directory

next the frontend server can be started by running "npm start"

finally visit "http://localhost:3000" to interact with this app
